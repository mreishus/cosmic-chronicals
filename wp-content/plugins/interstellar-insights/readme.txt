A cosmic "Hello World" plugin for testing purposes.

== Description ==

Interstellar Insights is a simple plugin that displays a cosmic greeting message. It's perfect for testing plugin installation and activation in the Cosmic Chronicals workshop environment.

== Installation ==

1. Upload the `interstellar-insights` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode [interstellar_insights] in your posts or pages to display the cosmic greeting
